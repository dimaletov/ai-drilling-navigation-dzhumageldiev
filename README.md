# Сравнительный анализ ИИ-ассистентов в задачах навигации и проводки скважин
**Автор:** Джумагелдиев М., группа БНГа-о24 (ЮРГПУ (НПИ))

---

## 1. Цель исследования
Целью данной работы является сравнительный анализ функциональных возможностей зарубежного (**Gemini**) и отечественного (**YandexGPT/Алиса**) чат-ботов с искусственным интеллектом. Исследование сфокусировано на решении высокотехнологичных задач бурения: управлении траекторией ствола и сопровождении геонавигации в режиме реального времени.

## 2. Структура репозитория
Для обеспечения прозрачности и удобства навигации по проекту данные распределены по следующим каталогам:
* `cases/` — подробное описание инженерных сценариев (кейсов).
* `prompts/` — тексты профессиональных запросов, сформированные для тестирования.
* `results/` — визуальные подтверждения (скриншоты) ответов нейросетей.
    * `foreign_bot/` — ответы модели Gemini (Google).
    * `domestic_bot/` — ответы модели YandexGPT (Яндекс).
* `analysis/` — подробный сравнительный анализ и итоговые выводы.

## 3. Выбранные сценарии (Кейсы)

### Кейс №1: Анализ выхода кривизны (ННБ)
* **Проблема:** Фактическая интенсивность набора угла ниже проектных значений при работе с ВЗД.
* **Задача для ИИ:** Провести диагностику причин отклонения от проекта и предложить корректирующие мероприятия по режимам бурения.

### Кейс №2: Геонавигационное сопровождение
* **Проблема:** Риск выхода траектории горизонтального участка в кровлю продуктивного пласта.
* **Задача для ИИ:** Разработать алгоритм действий инженера-геонавигатора по корректировке зенитного угла и междисциплинарному взаимодействию.

## 4. Методология
1. **Разработка запросов:** Созданы специализированные промпты, использующие терминологию БНГА (КНБК, ВЗД, гамма-каротаж, зенитный угол).
2. **Эксперимент:** Запросы отправлены обоим сервисам в идентичных временных рамках.
3. **Оценка:** Сравнение проводилось по критериям технической глубины, отсутствия "галлюцинаций" и практической ценности для ИТР.

## 5. Итоговые выводы
* **Gemini** продемонстрировал превосходство в анализе механики работы отклонителей и диагностике оборудования.
* **YandexGPT** показал высокую эффективность в выстраивании операционных процессов и регламентировании действий персонала согласно российским стандартам управления.
